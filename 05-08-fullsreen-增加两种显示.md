# 📱 AsiaFM 播放器界面设计规范

## 1. 设计概述
本项目旨在优化 AsiaFM 高清音乐台的移动端播放界面。核心原则是**“不增加新功能，仅优化视觉布局”**。
- **核心视觉元素**：AsiaFM HD 音乐台 Logo（圆形发光样式）。
- **功能限制**：不显示歌曲列表，不显示时间进度条。
- **适配目标**：针对竖屏（单手操作）和横屏（车载/平板/沉浸模式）进行差异化布局。

---

## 2. 竖屏显示规范 (Portrait Mode)

竖屏模式下，界面采用**垂直居中堆叠**布局，强调视觉重心和底部的操作便捷性。

### 布局结构
1.  **顶部信息区**：显示电台名称和当前歌曲信息。
2.  **核心视觉区**：屏幕正中央放置圆形 Logo，带有发光特效。
3.  **底部控制区**：悬浮式控制栏，包含播放/暂停及切歌按钮。

### 视觉示意
```text
+-------------------------------+
|  AsiaFM高清音乐台             |
|  许巍 - 我们                  |
|                               |
|                               |
|      ( 圆形发光 Logo )        |
|      [ ASIA FM ]              |
|      [ HD音乐台  ]            |
|                               |
|                               |
|  ---------------------------  |
|  [⏮]      [⏹]      [⏭]       |
|  ---------------------------  |
+-------------------------------+

````

### 样式细节

- **背景**：深蓝色渐变背景（参考 `#1a2b3c` 到 `#000000`）。
- **Logo**：居中显示，直径约占屏幕宽度的 60%，添加外发光（Glow）效果。
- **控制栏**：底部悬浮半透明胶囊状背景，白色图标。

***

## 3. 横屏显示规范 (Landscape Mode)

横屏模式下，为了解决“空旷”和“拉伸”的问题，采用**左右分栏**布局。利用横向空间，左侧展示视觉，右侧展示信息与操作。

### 布局结构

1. **左侧视觉区 (40%)**：放置 Logo，尺寸适中，垂直居中。
2. **右侧信息与控制区 (60%)**：
   - 上方：电台名与歌名（左对齐）。
   - 下方：播放控制按钮组。

### 视觉示意

```
+-------------------------------------------+
|                                           |
|   +-------------+   AsiaFM高清音乐台      |
|   |             |   ------------------    |
|   |  ( Logo )   |   许巍 - 我们           |
|   |             |                         |
|   +-------------+                         |
|                                           |
|               [⏮]   [⏹]   [⏭]             |
|                                           |
+-------------------------------------------+

```

### 样式细节

- **分割**：左右区域不需要明显的硬分割线，通过间距自然区分。
- **对齐**：右侧所有文字和按钮均**左对齐**，起始位置与左侧 Logo 的中心线或边缘保持视觉平衡。
- **Logo**：不再占据全屏高度，而是缩放到适合左侧区域的大小，保持圆形发光样式。
- **按钮**：按钮尺寸可以比竖屏略大，增加点击热区，适应横屏握持。
